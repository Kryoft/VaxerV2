INSTALLAZIONE

	- Installare PostgreSQL
	- Inserire i dati utente e password usati per Postgre sostituendo quelli compresi tra <username></username> e tra 
  	  <password></password> nel file pom.xml (righe 83, 98 e 113).
	- Cliccare due volte sul file maven-clean-install
	- Nella cartella jar saranno disponibili due cartelle: Client e Server, che contengono ciascuna
  	  il rispettivo file jar da far eseguire a client o server


SERVER:

	- Eseguire il file ServerCV

CLIENT:

	- Eseguire il file ClientCV



JAVADOC

	Eseguire il file maven-create-javadoc.bat.
	In seguito la javadoc sarà generata in doc/javadoc.
	Una javadoc è stata già generata in tale cartella.
	Per visualizzarla è sufficiente aprire il file index.html



Per ulteriori informazioni fare riferimento al manuale utente (Manuale_Utente.pdf).
